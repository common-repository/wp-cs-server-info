== Installation ==

Install

1: Unzip the `wp-cs-server-info.zip` file. 
2: Upload the the `wp-cs-server-info` folder to your `wp-contents/plugins` folder. 

Activate

1: In your WordPress Administration, go to the Plugins page
2: Activate the plugin `WP CS SERVER INFO`.
4: Navigate to the Widgets page, drag the `wp-cs-server-info` to the sidebar thats in use.
5: Enter the Name, Server IP & Port number, Click save.
6: visit your homepage (or were the sidebar will be loaded)and you should see the server info.

If you find any bugs or have any ideas, please mail me.
